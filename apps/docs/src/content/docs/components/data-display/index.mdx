description: Components for organizing and presenting information
---

Data display components help organize and present information in a clear, accessible manner. These components make complex data structures easy to navigate and understand.

## Available Components

- **[Tabs](/components/data-display/tabs/)** - Organize content into switchable panels
- **[Accordion](/components/data-display/accordion/)** - Collapsible content sections for space-efficient information display

## Design Principles

- **Information Hierarchy**: Clear visual hierarchy to guide user attention
- **Progressive Disclosure**: Show essential information first, details on demand
- **Responsive Design**: Adapts to different screen sizes and contexts
- **Keyboard Navigation**: Full keyboard accessibility for all interactive elements
