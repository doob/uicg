Layout components provide the structural foundation for your pages and applications. They help organize content and create consistent, responsive layouts.

## Available Components

- **[Hero](/components/layout/hero/)** - Prominent page headers with call-to-action elements
- **[Sidebar](/components/layout/sidebar/)** - Navigation and content organization panels

## Design Principles

- **Responsive Design**: Layouts adapt seamlessly to different screen sizes
- **Consistent Structure**: Predictable patterns that users can rely on
- **Content Focus**: Layout supports and enhances content without distraction
- **Performance**: Efficient rendering and minimal layout shifts
