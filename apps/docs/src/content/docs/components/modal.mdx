### Properties

##### `id` (required)

**type:** `string`

**default:** `dialog`

##### `intent`

**type:** `string`

**default:** `info`

**options**: `info`, `success`, `warning`, `danger`

##### `closeOutside`

**type:** `boolean`

**default:** `true`

Closes the modal when clicking outside.
